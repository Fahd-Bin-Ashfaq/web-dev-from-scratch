# web-dev-from-scratch

A complete **beginner-friendly 12-week roadmap** designed to help you learn the fundamentals of **HTML, CSS, JavaScript, Git, and website deployment** from scratch.

Throughout this journey, you will build multiple mini-projects and one final **responsive website**.  
By the end, you will be able to **create, style, and deploy** your own websites using free hosting platforms like:

- **GitHub Pages**
- **Netlify**
- **Vercel**

This repository includes:

- Weekly learning goals  
- Coding exercises  
- Best practices  
- Step-by-step deployment guides  

Start your web development journey with a clear path and practical hands-on experience!
