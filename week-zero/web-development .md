## 3. How the Web Works (Basics)

1. You enter a website URL (e.g., `google.com`) in your browser.  
2. The browser sends a request to the **web server**.  
3. Server processes the request and returns **HTML, CSS, JavaScript**.  
4. Browser displays the website.

### Real-Life Example
When you search something on Google:

- Your browser sends a request.  
- Googleâ€™s servers send back the results.  
- Browser displays them instantly.

## 5. Types of Web Development

### 1. **Static Websites**
- Content does not change dynamically  
- No database  
**Example:** Portfolio

### 2. **Dynamic Websites**
- Content changes based on interaction  
- Uses database + backend  
**Example:** Facebook, YouTube

### 3. **Single Page Applications (SPAs)**
Loads once, updates content dynamically.  
**Uses:** React, Vue, Angular  
**Example:** Gmail, Airbnb dashboard

### 4. **Progressive Web Apps (PWAs)**
Websites that feel like mobile apps.  
**Example:** Twitter Lite, Instagram Web

### 5. **E-commerce Development**
Building online stores.  
**Example:** Daraz, Amazon, Shopify stores

### 6. **APIs & Backend Services**
Connecting applications and providing data.  
**Example:** Easypaisa & JazzCash APIs
